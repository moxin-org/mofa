Hello World

最简单的入门示例，创建第一个 AI 代理

查看完整代码 
from mofa import Agent, Pipeline
from mofa.messages import HumanMessage

# 创建代理
agent = Agent(
    name="hello-agent",
    system_prompt="你是一个友好的助手"
)

# 创建管道并运行
pipeline = Pipeline().add(agent)
response = pipeline.run(
    HumanMessage("你好！")
)

print(response.content)
中级 难度: ★★★☆☆ 预计时间: 30分钟

RAG 问答系统

基于检索增强生成的智能问答系统

查看完整代码 
from mofa import Agent, Pipeline
from mofa.agents import RAGAgent
from mofa.knowledge import VectorStore

# 创建知识库
knowledge = VectorStore()
knowledge.add_documents([
    "MoFA 是一个模块化的 AI 代理框架",
    "它支持组合式开发模式"
])

# 创建 RAG 代理
rag_agent = RAGAgent(
    name="rag-qa",
    knowledge_base=knowledge
)

# 运行查询
pipeline = Pipeline().add(rag_agent)
response = pipeline.run("什么是 MoFA？")
高级 难度: ★★★★☆ 预计时间: 60分钟

Arxiv 论文分析

自动分析和总结学术论文的 AI 代理

查看完整代码 
from mofa import Agent, Pipeline
from mofa.agents import PaperAnalyzer
from mofa.tools import ArxivTool

# 创建论文分析代理
analyzer = PaperAnalyzer(
    name="arxiv-analyzer",
    tools=[ArxivTool()]
)

# 分析论文
pipeline = Pipeline().add(analyzer)
result = pipeline.run(
    "分析论文 arxiv:2301.00001"
)

print(result.summary)
高级 难度: ★★★★☆ 预计时间: 45分钟

反思代理

具有自我反思能力的智能代理系统

查看完整代码 
from mofa import Agent, Pipeline
from mofa.patterns import ReflectionPattern

# 创建反思代理
agent = Agent(
    name="reflection-agent",
    pattern=ReflectionPattern(
        max_iterations=3
    )
)

pipeline = Pipeline().add(agent)
response = pipeline.run(
    "写一个Python排序算法并优化它"
)

多代理协作

多个 AI 代理协同工作完成复杂任务

查看完整代码 
from mofa import Agent, Pipeline
from mofa.patterns import CollaborationPattern

# 创建多个专门代理
researcher = Agent(name="researcher")
writer = Agent(name="writer") 
reviewer = Agent(name="reviewer")

# 设置协作模式
pipeline = Pipeline(
    pattern=CollaborationPattern()
)
pipeline.add(researcher)
pipeline.add(writer)
pipeline.add(reviewer)

result = pipeline.run(
    "研究并写一篇关于 AI 发展的文章"
)
